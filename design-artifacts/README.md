# Design Artifacts

Architecture design outputs for the AZ-305 Budget Patrol project, organised by exam phase.

## Structure

- One folder per AZ-305 module
- Each folder contains SVG diagrams plus a `README.md`
- `README.md` tracks decisions, provisioned Azure resources, and open items
- SVGs are generated artifacts — do not edit manually

## Phases

| Folder | AZ-305 Phase |
|--------|-------------|
| `phase-1-identity-governance/` | Phase 1 — Identity & Governance |
| `phase-2-business-continuity/` | Phase 2 — Business Continuity |
| `phase-3-data-storage/` | Phase 3 — Data Storage |
| `phase-4-infrastructure/` | Phase 4 — Infrastructure |
| `phase-5-waf/` | Phase 5 — Well-Architected Framework |
